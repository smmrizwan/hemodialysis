# Dialysis Patient Management System

## Quick Setup
1. Upload all files to your web server
2. Make sure PHP 8+ is installed with SQLite support
3. Run: composer install (if composer available)
4. Access index.php in your browser

- Complete patient management system
- Advanced quarterly lab calculations
- Comprehensive reporting and exports
- SQLite database (no MySQL setup needed)
